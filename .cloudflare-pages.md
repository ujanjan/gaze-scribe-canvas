# Cloudflare Pages Deployment Quick Reference

## Build Configuration

- **Build command**: `npm run build`
- **Build output directory**: `dist`
- **Node.js version**: 18 or higher
- **Framework preset**: Vite (or "None" if not available)

## Environment Variables

Add the following environment variable in Cloudflare Pages dashboard:

- **Variable name**: `VITE_GEMINI_API_KEY`
- **Value**: Your Google Gemini API key (optional, for AI analysis features)

## Important Files

- `public/_redirects` - Ensures SPA routing works correctly (all routes redirect to index.html)

## Deployment Steps

1. Push code to Git repository (GitHub, GitLab, or Bitbucket)
2. In Cloudflare Dashboard → Pages → Create a project
3. Connect your Git repository
4. Configure build settings (see above)
5. Add environment variables (optional)
6. Deploy!

## Notes

- The `_redirects` file is automatically copied to the `dist` directory during build
- All routes will properly serve the React app
- Environment variables are available at build time for Vite
